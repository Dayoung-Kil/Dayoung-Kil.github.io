[Token Fusion / ToFu] Bridging the Gap between Token Pruning and Token Merging

WACV 2024

2023 В  В· В  hybrid В  В· В  token-reduction-in-vits

Hybrid WACV 2024

Minchul Kim, Shangqian Gao, Yen-Chang Hsu, Yilin Shen, Hongxia Jin В· Michigan State University / Samsung Research America

arXiv

н•ң мӨ„ мҡ”м•Ҫ. "mergingмқҙ н•ӯмғҒ pruningліҙлӢӨ лӮҳмқҖк°Җ?"лҘј л”°м ё, layerмқҳ functional linearity(мһ…л Ҙ ліҙк°„м—җ лҢҖн•ң м¶ңл Ҙмқҳ м„ нҳ•м„ұ)м—җ л”°лқј л‘ҳмқ„ к°Ҳм•„ м“ҙлӢӨ вҖ” linearityк°Җ лӮ®мқҖ мҙҲкё° layerлҠ” pruning, лҶ’мқҖ нӣ„кё° layerлҠ” merging(hybrid merging). лҳҗ average mergeк°Җ feature normмқ„ к№Һм•„ л¶„нҸ¬к°Җ нӢҖм–ҙм§ҖлҠ” л¬ём ңлҘј MLERP(norm ліҙмЎҙ нҸүк· )лЎң кі м№ңлӢӨ. н•ҷмҠө м—Ҷмқҙ ToMeліҙлӢӨ лҚ” м •нҷ•н•ҳкі  лҚ” л№ лҘҙлӢӨ(л¶„лҘҳВ·мқҙлҜём§Җ мғқм„ұ лӘЁл‘җ).

л°°кІҪ

ToMeмқҳ л“ұмһҘмңјлЎң merging(л№„мҠ·н•ң нҶ нҒ°мқ„ нҸүк· )мқҙ pruning(к·ёлғҘ лІ„лҰ¬кё°)мқ„ лҢҖмІҙн•ҳлҠ” л“Ҝ ліҙмҳҖлӢӨ. мқҙ л…јл¬ёмқҖ н•ң л°ң л¬јлҹ¬м„ң л¬»лҠ”лӢӨ вҖ” вҖңaverage mergingмқҙ м •л§җ н•ӯмғҒ pruningліҙлӢӨ лӮҳмқҖк°Җ?вҖқ

  • pruning vs mergingмқҳ ліём§Ҳ м°Ёмқҙ вҖ” pruningмқҖ кё°мЎҙ нҶ нҒ° н‘ңнҳ„мқ„ к·ёлҢҖлЎң лӮЁкёҙлӢӨ. mergingмқҖ л‘җ нҶ нҒ°мқ„ нҸүк·  лӮҙ мһ…л Ҙмқ„ ліҙк°„(interpolate) н•ңлӢӨ. к·ёлҹ°лҚ° л’Өл”°лҘҙлҠ” MLPВ·ATTNмқҖ л№„м„ нҳ•мқҙлқј, ліҙк°„лҗң мһ…л Ҙмқҳ м¶ңл Ҙмқҙ л‘җ мӣҗліё м¶ңл Ҙмқҳ мӨ‘к°„м—җ м•Ҳ лҶ“мқј мҲҳ мһҲлӢӨ вҶ’ м¶ңл Ҙ кіөк°„м—җм„ң м–ҙкёӢлӮҳ(misalignment) м •ліҙ мҶҗмӢӨВ·л¶„нҸ¬ мқҙлҸҷмқҙ мғқкёҙлӢӨ.
  • к·ё м •лҸ„к°Җ layerл§ҲлӢӨ лӢӨлҘҙлӢӨ вҖ” лё”лЎқмқҳ вҖңм„ нҳ•м„ұвҖқмқҙ лӮ®мқҖ кіім—җм„ң нҸүк· мқҖ мң„н—ҳн•ҳкі , лҶ’мқҖ кіім—җм„ң нҸүк· мқҖ м•Ҳм „н•ҳл©° мҳӨнһҲл Ө м—¬лҹ¬ нҶ нҒ° м •ліҙлҘј мў…н•©н•ҙ мқҙл“қмқҙлӢӨ.

нҸүк· (merge)мқҙ мўӢмқҖ layerмҷҖ к·ёлғҘ лІ„лҰ¬лҠ”(prune) кІҢ мўӢмқҖ layerк°Җ л”°лЎң мһҲлӢӨл©ҙ, layerлі„лЎң к°Ҳм•„ м“°л©ҙ лҗҳм§Җ м•Ҡмқ„к№Ң?

н•өмӢ¬ м•„мқҙл””м–ҙ

ToFuлҠ” pruningкіј mergingмқ„ н•ҳлӮҳмқҳ м•Ңкі лҰ¬мҰҳмңјлЎң мқјл°ҳнҷ”н•ңлӢӨ. нҶ лҢҖлҠ” ToMeмқҳ BSM(Bipartite Soft Matching) вҖ” нҶ нҒ°мқ„ SRCВ·DSTлЎң лӮҳлҲ  к°ҖмһҘ л№„мҠ·н•ң rмҢҚмқ„ м°ҫлҠ”лӢӨ. к°ҷмқҖ л§Өм№ӯ кІ°кіј мң„м—җм„ң м„ё к°Җм§Җ кІ°н•© л°©мӢқмқ„ кі лҘј мҲҳ мһҲкІҢ н•ңлӢӨ.

  • Average merging вҖ” srcлҘј dst мң„м№ҳм—җ нҸүк· (scatter-reduce mean). ToMeмҷҖ лҸҷмқј.
  • Pruned merging вҖ” л№„мҠ·н•ҳлӢҲ srcлҘј к·ёлғҘ лІ„лҰ°лӢӨ(dstлҠ” к·ёлҢҖлЎң). нҸүк·  м—°мӮ°мқҙ м—Ҷм–ҙ averageліҙлӢӨ л№ лҘҙлӢӨ.
  • MLERP merging вҖ” нҸүк· мқҙлҗҳ normмқ„ ліҙмЎҙ(м•„лһҳ м„ӨлӘ…).

м–ҙлҠҗ layerм—җ л¬ҙм—Үмқ„ м“ём§ҖлҠ” functional linearityлЎң м •н•ңлӢӨ.

л°©лІ•

Functional Linearity вҖ” м–ҙл””м„ң нҸүк· мқҙ мң„н—ҳн•ңк°Җ

н•ЁмҲҳ $f$ мҷҖ л‘җ нҶ нҒ° $X_1, X_2$ м—җ лҢҖн•ҙ, м¶ңл Ҙ мӮ¬мқҙмқҳ м§Ғм„  кұ°лҰ¬лҘј ліҙк°„ кІҪлЎңлҘј л”°лқјк°„ мӢӨм ң мқҙлҸҷ кұ°лҰ¬лЎң лӮҳлҲҲ к°’мңјлЎң м„ нҳ•м„ұмқ„ мһ°лӢӨ:

\[\text{FL}(f, X_1, X_2) = \frac{\| f(X_1) - f(X_2) \|_2}{\sum_{i=1}^{N-1} \Delta f(t_i)}, \qquad X(t) = (1-t)X_1 + tX_2\]

мҷ„м „нһҲ м„ нҳ•мқҙл©ҙ 1. мӮ¬м „н•ҷмҠө ViT-Sмқҳ к°Ғ block MLPлҘј ImageNet 5л§Ң мһҘмңјлЎң мёЎм •н•ҳлӢҲ, мҙҲкё° layerлҠ” FLмқҙ лӮ®кі  к№Ҡм–ҙм§ҲмҲҳлЎқ 1.0м—җ мҲҳл ҙн•ңлӢӨ. мҰү мҙҲкё°м—” нҸүк· мқҙ мң„н—ҳ, нӣ„кё°м—” м•Ҳм „.

Hybrid Merging вҖ” layerлі„ м „нҷҳ

мқҙ кҙҖм°°мқ„ к·ёлҢҖлЎң к·ңм№ҷмңјлЎң м“ҙлӢӨ. м „нҷҳ к№Ҡмқҙ $d$ лҘј л‘җкі :

\[\text{method} = \text{PRUNE if } l < d \text{ else AVG(MLERP)}\]

мҰү мҙҲкё° $d$к°ң layerлҠ” pruned merge, мқҙнӣ„лҠ” (M)LERP merge. ablationм—җм„ң $d=6$(ViT-B 12мёө кё°мӨҖ)мқҙ мөңм Ғ. мҲңм„ңк°Җ кІ°м •м Ғ вҖ” PRUNEвҶ’AVGк°Җ 80.17%мқёлҚ° кұ°кҫёлЎң AVGвҶ’PRUNEмқҖ 77.73%лЎң кёүлқҪн•ңлӢӨ. вҖңмҙҲкё° layerм—җм„ң мһ…л Ҙ кіөк°„ ліҙк°„мқ„ н”јн•ҳлқјвҖқлҠ” мӣҗм№ҷмқ„ м •лҹүм ҒмңјлЎң ліҙм—¬мӨҖлӢӨ.

MLERP вҖ” normмқ„ ліҙмЎҙн•ҳлҠ” нҸүк· 

average mergeмқҳ лҳҗ лӢӨлҘё м•Ҫм җ: $\lVert \frac{x_1+x_2}{2}\rVert \le \max(\lVert x_1\rVert, \lVert x_2\rVert)$ вҖ” нҸүк· мқҙ feature normмқ„ к№ҺлҠ”лӢӨ. normмқҖ лҸ„л©”мқё м Ғмқ‘В·style transferм—җм„ң ліҙл“Ҝ л¶„нҸ¬мқҳ н•өмӢ¬ нҶөкі„лқј, мӨ„м–ҙл“Өл©ҙ н•ҷмҠө л¶„нҸ¬м—җм„ң лІ—м–ҙлӮңлӢӨ(pruningмқҖ лӮЁмқҖ нҶ нҒ° normмқ„ к·ёлҢҖлЎң л‘ ). SLERP(кө¬л©ҙ ліҙк°„)лҠ” 2к°ң нҶ нҒ° м „мҡ©мқҙлқј лӢӨмӨ‘ нҶ нҒ°м—” лӘ» м“ҙлӢӨ. к·ёлһҳм„ң MLERP(Maximum-norm Linear intERPolation):

\[\bar x = \frac{1}{K}\sum_{k=1}^{K} x_k, \qquad \text{MLERP}(\{x_1\dots x_K\}) = \frac{\bar x}{\lVert \bar x\rVert} \times \max_{k} \lVert x_k\rVert\]

нҸүк· мқҳ л°©н–ҘмқҖ м“°лҗҳ нҒ¬кё°лҠ” к°ңлі„ нҶ нҒ° мӨ‘ мөңлҢҖ normмңјлЎң лҗҳлҸҢлҰ°лӢӨ. н•ҷмҠө м—Ҷмқҙ averageлҘј MLERPлЎң л°”кҫёкё°л§Ң н•ҙлҸ„ м„ұлҠҘмқҙ мҳӨлҘёлӢӨ.

м¶”к°Җ нҢҢлқјлҜён„°В·н•ҷмҠөмқҙ м „нҳҖ м—ҶлӢӨ вҖ” BSM л§Өм№ӯ мң„м—җ вҖңм–ҙлҠҗ layerм—җ prune/mergeлҘј м“ём§ҖвҖқмҷҖ вҖңmergeлҘј norm ліҙмЎҙнҳ•мңјлЎңвҖқ л‘җ к°Җм§Җл§Ң м–№мқҖ кІғ.

кІ°кіј

ImageNet л¶„лҘҳ(MAEВ·AugReg ViT) + Stable Diffusion мқҙлҜём§Җ мғқм„ұ. throughputмқҖ V100 FP32.

  • ToMeлҘј м •нҷ•лҸ„В·мҶҚлҸ„ лӘЁл‘җ м¶”мӣ” вҖ” ViT-B r=8: ToMe 82.86% вҶ’ ToFu AVG 83.19% вҶ’ MLERP 83.22%(full 83.74%). м••м¶•мқҙ м…ҖмҲҳлЎқ кІ©м°Ё нҷ•лҢҖ вҖ” r=20м—җм„ң ToMe 67.54% вҶ’ MLERP 74.06%(+6.5%p). ViT-LмІҳлҹј к№ҠмқҖ(24мёө) лӘЁлҚём—җм„ң нҡЁкіјк°Җ лҚ” нҒ¬лӢӨ.
  • л‘җ кё°м—¬мқҳ л¶„н•ҙ вҖ” ToMeвҶ’ToFu AVGлҠ” hybrid(мҙҲкё° prune) нҡЁкіј, ToFu AVGвҶ’MLERPлҠ” norm ліҙмЎҙ нҡЁкіј. л‘ҳ лӢӨ н•ҷмҠө м—Ҷмқҙ лҸҷмһ‘.
  • н•ҷмҠө м—Ҷмқҙ SOTA вҖ” AugReg r=13м—җм„ң 79.6%, к°ҷмқҖ 2.7 GFLOPsмқҳ DynamicViT(79.3, н•ҷмҠө н•„мҡ”)В·ToMe(79.3~79.4)ліҙлӢӨ лҶ’лӢӨ. A-ViTВ·DynamicViTВ·SP-ViTк°Җ auxiliary н•ҷмҠөмқҙ н•„мҡ”н•ҳкі  batch л№„нҳёнҷҳмқё л°ҳл©ҙ, ToFuлҠ” м¶”к°Җ нҢҢлқјлҜён„° 0В·batch нҳёнҷҳ.
  • мқҙлҜём§Җ мғқм„ұ(Stable Diffusion) вҖ” ToMeSDмІҳлҹј ATTN м•һм—җ лҒјмӣҢ 50% к°җм¶•. ToMe лҢҖл№„ FID 15.74вҶ’14.72, LPIPS 0.313вҶ’0.271, MS-SSIM 0.730вҶ’0.762лЎң м „л¶Җ к°ңм„ . мҙҲкё° layer ATTNмқҙ мқҙлҜём§Җмқҳ м „мІҙ кө¬мЎ°лҘј мһЎлҠ”лҚ°, кұ°кё°м„ң pruned mergeк°Җ кө¬мЎ°лҘј лҚ” мһҳ ліҙмЎҙн•ҳкё° л•Ңл¬ё(ліҙк°„мқ„ н”јн•Ё).

н•ң мӨ„ м •лҰ¬ & мқҳмқҳ

  • pruningкіј mergingмқ„ лҢҖлҰҪмқҙ м•„лӢҲлқј layerлі„лЎң к°Ҳм•„ м“°лҠ” Hybrid кі„м—ҙ. вҖңmergingмқҖ мһ…л Ҙ ліҙк°„мқҙлқј л№„м„ нҳ• layerм—җм„ң мң„н—ҳвҖқмқҙлқјлҠ” functional linearity л¶„м„қмңјлЎң, мҙҲкё°=pruneВ·нӣ„кё°=mergeлқјлҠ” к·ңм№ҷмқ„ м •лӢ№нҷ”н•ңлӢӨ.
  • л‘җ к°ңмқҳ к°ҖлІјмҡҙ к°ңм„  вҖ” (1) hybrid(pruneвҶ’merge мҲңм„ң), (2) MLERP(norm ліҙмЎҙ нҸүк· ). л‘ҳ лӢӨ BSM мң„м—җ м–№лҠ” л¬ҙн•ҷмҠө мһҘм№ҳлқј ToMeлҘј к·ёлҢҖлЎң лҢҖмІҙ к°ҖлҠҘ.
  • мң„м№ҳ. DiffRateк°Җ pruneВ·mergeлҘј н•ң layer м•Ҳм—җм„ң л№„мңЁлЎң м„һкі  к·ё л№„мңЁмқ„ н•ҷмҠөн–ҲлӢӨл©ҙ, ToFuлҠ” layer лӢЁмң„лЎң л‘ҳ мӨ‘ н•ҳлӮҳлҘј нҶөм§ёлЎң кі лҘҙкі  к·ё кё°мӨҖ(linearity)мқ„ н•ҷмҠө м—Ҷмқҙ м •н•ңлӢӨ. mergingмқҳ norm мҶҗмӢӨмқ„ м§ҡмқҖ м җмқҖ мқҙнӣ„ лӢӨкё°мӨҖ fusion([MCTF]) нқҗлҰ„кіјлҸ„ л§һлӢҝлҠ”лӢӨ. вҶ’ Token Reduction к°ңмҡ”
